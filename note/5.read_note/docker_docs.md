什么是容器？
	是一个沙盒程序在本地机器上运行，和其他本地机器上运行的程序相互隔离。这种隔离通过linux内核的命名空间和cgroups实现。这是Linux很久前就实现的功能，docker旨在将这些能力变得更可接触和更可用。

什么是容器的镜像？
	当容器运行时，它会使用一个隔离的文件系统。这个用户自定义的文件系统就是由容器镜像提供的。
