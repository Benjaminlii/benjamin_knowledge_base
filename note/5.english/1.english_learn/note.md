# English Learn Note

## 1. phrase 短语

```
no worries          别客气;太好了
by any chance       万一 碰巧 或许
looking forward to  期待
in order to         为了...
rely on             依赖
other than that     除此之外
```

## 2. abbreviation 缩写

```
AC (Accept criteria)            验收标准
TC (Test Case)                  测试案例
CI (Continuous Integration)     持续集成
CD (Continuous Delivery)        持续交付
```

## 3. sentence 句子

```
// 如果我错了请指正我。
Please correct me if I am wrong. 
```
